HTML : define the content of web pages
CSS : specify the layout of web pages
JavaScript : to program the behavior of web pages

JS can be applied either in the <head> tag or in < body> tag.
There are two ways to insert JS in HTML:
    - External .js file : <script src="location of external js file"></script>
    - In HTML page using <script> tag.

Best way is to place JS in a different file.
Because : It seperate HTML & JS and the JS file is chached and boost the loading speed.

We can see the output of JS in several ways:
    - To show calculated output in a HTML tag : innerHTML().
    - Directly write content using script tag : document.write().
    - Display the content in browser console : console.log().
    - Popup the JS content as a alert : window.alert().

All JS identifier are case sensitive:
ex: var name = "" and var Name = "" are different.

In JS the same variable can hold multiple data types:
    var x; (undefined)
    x = "str" (string)
    x = true (boolean)

Data types:
    Primitive data types: number, string, boolean, undefined
    Complex data types: functions and objects

JS Strings:
Zero or more characters written inside quotes are called strings, it uses for storing and manipulating text.

Methods in strings:
var str = "It is a string It is a string"

    Method name         Description                                         Example                             Output

1. length               Length of the string                                str.length                            29
2. indexOf()            First occurance of given string                     str.indexOf('is')                     3
                        second parameter define where to start              str.indexOf('a',8)                    21
3. lastIndexOf()        Last occurance of given string                      str.lastIndexOf('is')                 18
4. search()             Return the index of the match                       str.search('is')                      3

Difference between indexOf() and search():
    - The search() method cannot take a second start position argument.
    - The indexOf() method cannot take regular expressions.

Extracting the string:
    - slice() : It takes 2 parameters, the start position, and the end position (end not included).
        * str.slice(7, 13);             *str.slice(7); 
        * str.slice(-12, -6);           *str.slice(-12); 

    - substring() : Same as above but it does not take negative values as parameaters.
        * str.substring(7, 13);         * str.substring(7);

    - substr() : same as above,the difference is that the second parameter specifies the length of the extracted part.
        * str.substr(6,5);             * str.substr(6);

5. replace()        Replace first occurance of given string                 str.replace('string','sentence')    It is a sentence It is a string
6. toUpperCase()    Convert the string in upper case                    
7. toLowerCase()    Convert the string in lower case
